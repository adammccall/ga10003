Security Controls

How to use the security controls

To access the security controls, first you must click on the security button at the bottom of the screen:



The following options can be selected:

Lock Meeting

This allows you to close off the meeting and not even allow people to join the waiting room.

Enable waiting room

This is greyed out as you cannot disable the meeting room

Hide profile pictures

This turns off all profile images. If someone has their video off, their profile pic will not show.

Allow participants to:

Share Screen

Allows the paricipants to share screen or not.

Chat Disable or enable the chat function.
Rename themselves

Disable or enable the user to be able to change thier name after entering the meeting.

Start Video

Allow the participants to start and stop their video.

Suspend Participant Activities

When you press this, this will open up another dialog box (see right screen). Pressing suspend will lock down the meeting to allow you time to sort out any problems.
